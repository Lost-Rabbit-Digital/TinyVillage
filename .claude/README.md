# Claude Code Commands for Tiny Village

This directory contains custom slash commands to accelerate development of Tiny Village, a free-to-play cozy village-building game.

## Available Commands

### Core Development
- `/add-building` - Create a new building type with all necessary components
- `/create-biome` - Generate a new biome with terrain, resources, and atmosphere
- `/design-ui` - Design or improve UI elements following cozy game principles
- `/test-feature` - Create comprehensive test cases for any feature

### Monetization
- `/implement-dlc` - Set up DLC infrastructure and implement DLC content packs
- `/add-cosmetic` - Create new cosmetic items for the in-game shop
- `/cozy-review` - Review code/features against cozy game design principles

### Project Management
- `/plan-sprint` - Plan a development sprint based on the game design roadmap
- `/optimize-performance` - Analyze and optimize performance for smooth gameplay
- `/write-lore` - Create lore fragments and story content for the Khoswa-Kian

## Usage

In your Claude Code session, simply type the command:

```
/add-building
```

Claude will guide you through the process, asking for necessary details and implementing the feature according to the game design document principles.

## Design Principles

All commands enforce Tiny Village's core pillars:
1. **Cozy & Accessible** - Low-stress, intuitive, charming
2. **Creative Expression** - Flexible, decorative, shareable
3. **Meaningful Progression** - Rewarding without grinding
4. **Simple Depth** - Easy to learn, engaging to master

## Ethical Monetization

Commands related to DLC and microtransactions enforce ethical F2P principles:
- No pay-to-win mechanics
- Transparent pricing
- Optional content only
- Free alternatives always available
- No predatory dark patterns
